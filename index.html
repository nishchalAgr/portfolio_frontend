> This portfolio site itself is a fullstack project. The 'projects' tab is auto-generated by a custom Flask web service finds all github pages for a given user. Because this functionality is not included in the native Github API, it can be quite slow. Thus, to improve the responsiveness of the site, the web service performs hourly updates to a MongoDB database which can then act as a cache. Finally, the front end is also written from scratch.
